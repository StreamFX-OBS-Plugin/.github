## 🚀 Overview
StreamFX for Windows is a free, open-source plugin that expands what OBS Studio can do out of the box. It plugs directly into your existing OBS installation and unlocks a rich collection of sources, filters, and transitions — from real-time blur and 3D transform to advanced masking and custom shaders. Streamers, video creators, and podcasters use StreamFX to add polish and depth to their scenes without leaving OBS or reaching for a separate editing tool.

## ✨ Features
- **Blur filter** — Apply Gaussian, box, and directional blur to any source to soften backgrounds, hide sensitive details, or create depth-of-field looks.
- **3D transform** — Rotate, tilt, and reposition sources in three-dimensional space to build dynamic, layered scene layouts.
- **Advanced masking** — Combine sources with region, image, and gradient masks for clean cutouts and creative compositing.
- **Dynamic mask** — Drive one source's visibility or color from the properties of another for reactive, motion-aware effects.
- **Custom shaders** — Load your own shader code to create unique real-time effects, color grades, and animated overlays.

## 🎯 Getting Started
1. Make sure OBS Studio is installed and closed before you run the StreamFX installer.
2. Download and install StreamFX, then reopen OBS Studio.
3. To add a filter, right-click a source, choose **Filters**, and select a StreamFX effect such as Blur or 3D Transform.
4. To add a StreamFX source, click the plus icon in the Sources panel and pick one of the new entries.
5. Adjust the effect settings in the properties panel and watch your scene update live in the preview.

## ❓ FAQ
**Is StreamFX free?**
Yes. StreamFX is a free, open-source OBS plugin, so you can download and use it at no cost.

**Which versions of Windows are supported?**
StreamFX runs on 64-bit Windows 10 and Windows 11, alongside a supported version of OBS Studio.

**Is my data secure?**
StreamFX runs locally inside OBS Studio and processes your video on your own machine, so your scenes and recordings stay on your computer.
